# Sanity-Level-MOD

正気度の概念をマイクラに導入するFabric用Mod  
A Fabric mod that introduces the concept of sanity to Minecraft.

※The English version of the README is below

---

## 概要 (Overview)
プレイヤーに正気度ゲージ、調整用アイテムが複数追加されます。

- 正気度
  
特定のバイオーム、 ディメンションに居ること、  
暗闇に居ること、  
敵モブが近くにいることなどの要因で減少し、  
  
花畑などの特定バイオームに居ること、  
日光に当たっていること、  
オオカミや猫などのテイム済みモブが存在することで回復します。  

- ペナルティ
  
正気度が減少すると、  
視界が悪くなる(モノクロ化、ノイズなど)、  
幻聴(矢の飛来音など)、  
アイテムの自動ドロップなどのペナルティが課されます。  
正気度が0になった場合継続的にダメージを受けます。

## English Version

### Overview
This mod introduces a Sanity (Mental Health) system to Minecraft.

### Mechanics
- **Sanity Levels**
  - **Decrease**: Staying in dark areas, hostile dimensions/biomes, or being near monsters.
  - **Recovery**: Staying in peaceful biomes (e.g., Plains), being in sunlight, or staying near tamed pets (Dogs, Cats).

- **Penalties**
  As Sanity drops, players experience:
  - Visual distortions (Monochrome shaders, noise)
  - Auditory hallucinations (Ghostly sounds)
  - Involuntary item dropping
  - **Death**: Taking continuous damage when Sanity reaches 0.

### Featured Items
- **Caramel**: Restores 1 Hunger.
- **Milk Caramel**: Restores both Hunger and Sanity, granting a brief "Calm" effect.
- **Knot of Serenity**: A powerful end-game item that negates all Sanity loss while held in the inventory.
- **Potion of Sanity**: Brewed using Glowstone Dust with an Awkward Potion. Provides the "Calm" effect.
